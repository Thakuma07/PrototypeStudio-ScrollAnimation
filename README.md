# Prototype Studio Scroll Animation Showcase

A high-end, immersive web experience featuring smooth scroll-triggered animations. This project demonstrates advanced front-end techniques to create a visually stunning portfolio-style showcase for creative works.

## âœ¨ Key Features

- **Dynamic Project Indexing**: Real-time updating of the project counter (e.g., `01/10`) as you scroll.
- **Scroll-Triggered Image Reveal**: Images activate and reveal their full detail as they pass through the central spotlight area.
- **Smooth Content Panning**: Text and imagery move at different relative speeds to create a sense of depth and fluidity.
- **Full Smooth Scrolling**: Integration with Lenis for a consistent, premium feel across all devices.
- **Mobile Responsive**: Carefully crafted layouts that adapt seamlessly to smaller screens.
- **High Performance**: Optimized GSAP animations with proper use of `will-change` for hardware acceleration.
